# NITTC Scheduler

Android (Jetpack Compose + Material 3) で作成した時間割管理アプリです。

## 機能
- A/B/休日をタップで切替できる A/B 日表
- 期間内の平日を自動作成、土日・日本の祝日を自動で休日化
- 夏休み/冬休みなどの長期休み範囲登録（登録範囲は休日として反映）
- 1/2・3/4・5/6・7/8校時の時間割入力
- 毎週授業 / 隔週(A/B)授業の設定
- 授業名と担当教員名の入力
- 当日表示と端末カレンダー出力（今週のみ / 期間すべて）

## 開き方
1. Android Studio でこのフォルダを開く
2. JDK 17 以上を使って Gradle Sync
3. 実機またはエミュレータで実行

## パッケージ名
`jp.linkserver.nittcsc`
